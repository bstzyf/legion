## Task 1: Verify Registry-to-Disk Completeness

Validate that every agent listed in the registry catalog has a corresponding file on disk, and that every file on disk is listed in the registry.

**Actions:**

1. Extract all agent file paths from `skills/agent-registry/SKILL.md` Section 1 catalog tables
   - Parse every row matching the pattern `| {id} | \`agents/{id}.md\` |`
   - Collect the full set of registry paths

2. List all `.md` files in the `agents/` directory

3. Compare the two sets:
   - Registry paths that have no matching file on disk → **ERROR: phantom registry entry**
   - Files on disk that have no matching registry entry → **ERROR: unregistered agent**
   - Matching count should be exactly 51 in both directions

4. Verify each agent file has valid frontmatter:
   - Check each file in `agents/` has `name:` and `description:` in its YAML frontmatter
   - Check each file has a `division:` field

<verify>
```bash
# Count registry entries
REGISTRY_COUNT=$(grep -c '| `agents/' skills/agent-registry/SKILL.md)
echo "Registry entries: $REGISTRY_COUNT"

# Count files on disk
DISK_COUNT=$(ls agents/*.md | wc -l)
echo "Files on disk: $DISK_COUNT"

# Both should be 51
test "$REGISTRY_COUNT" -eq 51 && test "$DISK_COUNT" -eq 51 && echo "PASS: 51/51 match" || echo "FAIL: count mismatch"

# Cross-check: extract registry paths, verify each exists
grep -o '`agents/[^`]*`' skills/agent-registry/SKILL.md | tr -d '`' | sort -u | while read path; do
  test -f "$path" || echo "MISSING: $path"
done

# Cross-check: verify each disk file is in registry
for f in agents/*.md; do
  grep -q "$(basename $f)" skills/agent-registry/SKILL.md || echo "UNREGISTERED: $f"
done

# Check frontmatter fields
for f in agents/*.md; do
  head -20 "$f" | grep -q "^name:" || echo "NO NAME: $f"
  head -20 "$f" | grep -q "^description:" || echo "NO DESCRIPTION: $f"
  head -20 "$f" | grep -q "^division:" || echo "NO DIVISION: $f"
done
```
</verify>

**Done when:** All 51 registry entries map to existing files, all 51 files are registered, all have valid frontmatter. Zero errors from the cross-check.
