**Section 2: Campaign Planning Workflow (MKT-01)** — Structured campaign creation:

```
### 2.1: Campaign Brief Questioning
When a marketing phase is detected during /agency:plan, gather campaign parameters
using AskUserQuestion. This replaces the generic decomposition questioning for
marketing phases.

Key questions (adapt based on responses — don't ask all if answers imply others):

Q1: "What's the primary objective for this campaign?"
  Options:
  - "Brand awareness / Launch announcement"
  - "User acquisition / Growth"
  - "Community building / Engagement"
  - "Product promotion / Conversion"

Q2: "Who is your target audience?"
  Free text — capture demographics, interests, platforms they use

Q3: "Which channels should we target?" (multi-select)
  Options from DEFAULT_CHANNELS, filtered to those with available agents

Q4: "What's the campaign timeline?"
  Options:
  - "Sprint (1-2 weeks)" — high intensity, few channels
  - "Standard (4-6 weeks)" — full campaign cycle
  - "Ongoing (3+ months)" — sustained presence

Q5: "What's the key message or value proposition?"
  Free text — this becomes the core message for cross-channel coordination

### 2.2: Campaign Document Generation
After brief questioning, generate the campaign document at:
  {CAMPAIGN_DIR}/{campaign-slug}.md

The slug is derived from the campaign name: lowercase, spaces to hyphens, max 40 chars.
Example: "Product Launch Q2" → "product-launch-q2"

Use the format from Section 5. Populate all sections from the brief answers:
- Objectives: from Q1 + follow-up
- Target Audience: from Q2
- Channel Strategy: from Q3 + channel-agent mapping (Section 1)
- Core Messaging: from Q5
- Timeline: from Q4 + default phase structure
- Content Calendar: generate using Section 3 patterns
- Agent Assignments: from channel selection + agent-registry Section 4

### 2.3: Agent Team Assembly
For marketing campaigns, use the agent-registry Section 4 team assembly pattern:

Required roles:
- **Strategy Lead** (1): marketing-social-media-strategist
  Responsibilities: overall strategy, cross-channel alignment, campaign brief ownership
- **Content Lead** (1): marketing-content-creator
  Responsibilities: core content production, editorial calendar, brand voice consistency

Channel-specific roles (1 per selected channel):
- Map each selected channel to its primary agent using the Channel-Agent Mapping table
- If a channel's primary agent is already assigned (e.g., content-creator for Blog),
  no duplicate — the agent covers both roles

Optional roles:
- **Growth/Analytics**: marketing-growth-hacker
  Add when campaign objectives include "acquisition" or "conversion"
- **Visual Design**: design-visual-storyteller (cross-division)
  Add when campaign is visually intensive (Instagram, TikTok focus)
- **Analytics**: support-analytics-reporter (cross-division)
  Add when campaign requires detailed performance tracking

### 2.4: Campaign Lifecycle
Each campaign follows this lifecycle:
  Planning → Active → Measuring → Complete

Planning: Campaign document created, content calendar drafted, agents assigned
Active: Content being produced and distributed across channels
Measuring: Campaign ended, collecting metrics and learnings
Complete: Final report written, outcomes recorded to memory (if memory layer active)
```

**Section 3: Content Calendar Generation (MKT-02)** — Time-based content planning:

```
### 3.1: Calendar Structure
Content calendars live inside the campaign document (## Content Calendar section).
Format: weekly grid with channel assignments.

| Week | Phase | Key Theme | Content Items |
|------|-------|-----------|---------------|
| W1 | Pre-launch | Teaser/Anticipation | Twitter thread, Instagram teaser, Blog draft |
| W2 | Launch | Core announcement | All channels coordinated launch |
| W3-4 | Sustain | Engagement/Value | Regular cadence per channel |
| W5+ | Optimize | Best performers | Double down on top content |

### 3.2: Content Type Taxonomy
Each content item maps to a type, channel, and responsible agent:

| Type | Channels | Typical Agent | Cadence |
|------|----------|---------------|---------|
| Thread/Carousel | Twitter, LinkedIn | marketing-twitter-engager | 2-3x/week |
| Short-form Video | TikTok, Instagram Reels | marketing-tiktok-strategist | 2-3x/week |
| Stories | Instagram | marketing-instagram-curator | Daily during active |
| Long-form Post | Blog, Medium | marketing-content-creator | 1-2x/week |
| Email | Newsletter | marketing-content-creator | Weekly |
| Community Post | Reddit | marketing-reddit-community-builder | 2-3x/week |
| Poll/Interactive | Twitter, Instagram | platform-specific agent | Weekly |
| AMA | Reddit | marketing-reddit-community-builder | 1x per campaign |

### 3.3: Scheduling Patterns
Three standard patterns based on campaign timeline (from Q4):

**Sprint (1-2 weeks)**:
- High frequency: daily content on primary channels
- Focus: 2-3 channels max
- Pre-create all content before launch
- No optimization phase — measure after sprint ends

**Standard (4-6 weeks)**:
- Week 1: Teaser content (2-3 posts)
- Week 2: Launch burst (daily across all channels)
- Weeks 3-4: Sustain cadence (3-4x/week per channel)
- Week 5: Optimization (boost top performers)
- Week 6: Measurement and learnings

**Ongoing (3+ months)**:
- Month 1: Establish presence and baseline metrics
- Month 2: Optimize based on data
- Month 3+: Sustained cadence with monthly themes
- Review and adjust monthly

### 3.4: Assignment Rules
- Each content item has exactly ONE responsible agent (no shared ownership)
- Agent assignment follows the Channel-Agent Mapping (Section 1)
- Strategy Lead reviews all content for brand consistency (not creation, just review)
- Content Creator handles all long-form regardless of distribution channel
- Channel specialists own platform-native formats (Reels, TikToks, Threads)
```

**Section 4: Cross-Channel Coordination (MKT-03)** — Consistent messaging:

```
### 4.1: Core Message Derivation
Every campaign has ONE core message. Derived from:
- Campaign objective (Q1)
- Key value proposition (Q5)
- Target audience (Q2)

Format:
  Core Message: {one sentence that captures the campaign's central idea}
  Supporting Points:
    1. {rational benefit — what the audience gains}
    2. {emotional benefit — how it makes them feel}
    3. {social proof — why others trust/use it}

### 4.2: Channel Adaptation Framework
Same core message, adapted for each channel's norms:

| Channel | Tone | Format | Length | CTA Style |
|---------|------|--------|--------|-----------|
| Twitter | Conversational, punchy | Thread or single tweet | 280 chars or 5-7 tweet thread | "Reply with..." or link |
| Instagram | Aspirational, visual | Caption + visual | 150-300 words | "Link in bio" or "DM us" |
| TikTok | Casual, authentic | Script + visual notes | 15-60 second script | "Follow for more" or link in bio |
| Reddit | Informative, value-first | Text post or comment | 200-500 words | Subtle, never salesy |
| Blog | Authoritative, detailed | Article structure | 800-2000 words | In-text CTA |
| Email | Personal, direct | Newsletter format | 200-400 words | Button CTA |

### 4.3: Consistency Validation Checklist
Before Wave 2 (content creation) begins, verify:
- [ ] All channel agents have received the core message and supporting points
- [ ] Campaign hashtags are consistent across channels
- [ ] Visual style guidelines are shared (if applicable)
- [ ] CTA destinations are aligned (all point to same landing page/action)
- [ ] Tone guidelines per channel are documented in the campaign document
- [ ] Launch timing is coordinated (stagger by 2-4 hours across channels for max reach)

### 4.4: Cross-Channel Handoff Pattern
Strategy Lead (marketing-social-media-strategist) produces:
  → Core messaging brief (Section 4.1 format)
  → Channel adaptation guidelines (Section 4.2)
  → Timing coordination plan

Each Channel Agent receives:
  ← Core messaging brief (as context in their execution prompt)
  ← Channel-specific adaptation guidelines
  ← Content calendar assignments for their channel
  → Produces channel-specific content following the adaptation guidelines

Content Creator (marketing-content-creator) produces:
  → Long-form content (blog posts, email copy, landing page copy)
  → Content that channel agents can excerpt/reference for shorter formats

### 4.5: Coordination During Execution
During /agency:build for marketing phases:
- Wave 1 agent (Strategy Lead) produces the core messaging brief
- Wave 2 agents receive Wave 1 output as context (via SUMMARY.md)
- All Wave 2 agents get the same core messaging brief + their channel-specific instructions
- /agency:review for marketing checks cross-channel consistency as a review criterion
```
